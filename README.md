Project Description:
This system allows Linda to safely enter her home during winter evenings by automatically turning on the porch and hallway lights when a push button is pressed. The porch light stays on for 30 seconds and the hallway light for 60 seconds.

How the code works:
setup() initializes pins.
loop() continuously checks the button state and updates lights.
turnLightsOn() and updateLights() modularly manage the lights, improving readability and maintainability.

Instructions:
switch on the button to turn both lights on.
Porch light will turn off after 30 seconds, hallway after 60 seconds.
